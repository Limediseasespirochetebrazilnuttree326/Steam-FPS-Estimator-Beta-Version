## 🛠️ Using the estimator

The application features a simple interface. You do not need technical skills to get results. 

### Importing your library
When you open the tool for the first time, it asks for permission to scan your Steam library. Click "Allow" to let the software read your installed games. It pulls data from your PC hardware to build a base profile. This process takes less than one minute. 

### Running a prediction
Select a game from the list on the left side of the screen. The tool displays your estimated frame rate in the main window. You will see two numbers: a minimum expected frame rate and an average expected frame rate. 

### Adjusting settings
You see a list of graphics options for your chosen game. Each option has a slider. Move the slider to change the graphics quality. The software updates your estimated frame rate in real-time as you move the sliders. If the frame rate number stays low, the tool highlights settings you should turn down.

## 🔍 Understanding the data

The application uses local hardware benchmarks to calculate these numbers. It checks your GPU memory and your processor speed. This data stays on your machine. We do not transmit your hardware information to external servers. The tool only uses the data to calculate the difference between your current hardware and the performance requirements of the chosen game.
